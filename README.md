# Ohjelmistojen ylläpito ja testaus - Harjoitus 9

## Projektin kuvaus
Tämä harjoitustyö keskittyy yksikkötestaukseen, jatkuvaan integraatioon (CI) ja testikattavuuden seurantaan. Projektissa käytetään JavaScript-kirjastoa, jolle on kirjoitettu testit Jestiä hyödyntäen.

## Teknologiat
* **Testauskehikko:** [Jest](https://jestjs.io/)
* **Automaatio:** [GitHub Actions](https://github.com/features/actions)
* **Kattavuusraportointi:** [Coveralls](https://coveralls.io/)
* **Kääntäjä:** Babel

## Testien ajaminen paikallisesti
```bash
npm install
npm test
npm run coverage
